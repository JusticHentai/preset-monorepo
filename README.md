# Preset-monorepo

持续集成常用开发配置 monorepo

## Prettier

**dev：**
- `prettier`：格式化所有文件
- `@douza/prettier-config`：[详细](https://github.com/JusticHentai/prettier-config)

## Editor

**file：**
- `.editorconfig`：告知 ide 相关规则

## Typescript

**file：**
- `tsconfig.json`：配置 ts 规则

**dev：**
- `typescript` ts 本体
- `@types/react`：`react` 一套
- `@types/react-dom`：同上

## Eslint

**dev**
- `eslint` eslint 本体
